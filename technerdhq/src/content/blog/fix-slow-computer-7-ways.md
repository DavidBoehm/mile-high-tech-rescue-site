## 1. Clean Up Your Startup Programs

### What's Slowing You Down

Every time your computer boots up, dozens of programs might be launching in the background—many of which you don't even use. These startup apps hog memory and CPU power before you even open your browser.

### How to Fix Slow PC Startup

**Windows:**
1. Press `Ctrl + Shift + Esc` to open Task Manager
2. Click the "Startup" tab
3. Disable anything you don't need immediately on boot (Spotify, Adobe Updater, etc.)
4. Right-click → "Disable"

**Mac:**
1. Go to System Preferences → Users & Groups
2. Click "Login Items"
3. Select programs you don't need and click the minus (-) button

> **Pro Tip:** Keep your antivirus and cloud sync apps enabled. Everything else? Probably safe to disable.

## 2. Free Up Disk Space (Seriously, Do It)

### The 15% Rule

Your computer needs breathing room. When your hard drive is over 85% full, everything slows down—file access, app launches, even web browsing. If you're wondering how to **fix slow PC** issues, this is often the culprit.

### Quick Wins for Disk Cleanup

- **Empty your Recycle Bin/Trash** (you'd be surprised)
- **Uninstall unused programs** (Control Panel → Programs → Uninstall)
- **Clear browser cache** (Settings → Privacy → Clear browsing data)
- **Delete large files** you don't need (old videos, duplicate photos)
- **Use Disk Cleanup** (Windows) or **Optimize Storage** (Mac)

## 3. Scan for Malware and Viruses

### The Hidden Performance Killer

Malware doesn't always announce itself with pop-ups. Sometimes it just sits there, quietly draining your system resources, mining cryptocurrency, or running background processes that make your **computer running slow** feel like an understatement.

### What to Do

1. Run a full system scan with Windows Security (Windows Defender) or your antivirus
2. Consider a second-opinion scan with [Malwarebytes](https://www.malwarebytes.com) (free version works great)
3. Remove any suspicious browser extensions
4. Check for unfamiliar programs in your installed apps list

> **Warning:** If your browser's homepage changed or you're seeing weird ads, you might have adware. We can clean that up for you—[schedule a malware removal](https://technerdhq.milehightechrescue.com).

## 4. Update Everything (Yes, Everything)

### Why Updates Matter

Outdated software isn't just a security risk—it's often a performance drag. Updates include bug fixes, optimizations, and better resource management that can **speed up computer** performance significantly.

### Your Update Checklist

- **Operating System:** Windows Update or macOS Software Update
- **Graphics drivers:** Essential for gaming and video work
- **Browser:** Chrome, Firefox, Edge—keep them current
- **Key apps:** Office, Creative Suite, etc.
- **BIOS/UEFI:** Only if you're comfortable, or ask a pro

## 5. Add More RAM

### When Software Fixes Aren't Enough

If your computer is still sluggish after cleanup, you might be hitting a memory ceiling. Modern web browsing, video calls, and multitasking are memory-hungry activities. 4GB of RAM just doesn't cut it anymore.

### Quick Check

- Open Task Manager (Windows) or Activity Monitor (Mac)
- Check memory usage under heavy use
- If you're consistently at 80-90%+ RAM usage, it's upgrade time

## 7. Reset or Reinstall Windows/macOS

### The Nuclear Option (In a Good Way)

Sometimes, after years of use, computers just get… gunked up. Too many uninstalls leaving traces, registry clutter, background processes you can't identify—it's like digital hoarding. A fresh OS install can restore that "new computer" feel.

### Before You Reset

- **Back up your data** (seriously, this is critical)
- Make a list of programs you'll need to reinstall
- Gather license keys for paid software
- Consider keeping documents but doing a "clean" install

**Windows:** Settings → System → Recovery → Reset this PC
**Mac:** Restart in Recovery Mode (`Cmd + R`) → Reinstall macOS
